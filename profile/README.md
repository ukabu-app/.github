# UKABU - 就職活動管理プラットフォーム

就活生向けの採用メール自動整理・タスク管理アプリケーション

## 概要

UKABUは、日本の就職活動生が複数企業からの採用関連メールを効率的に管理できるプラットフォームです。
AIを活用し、受信したメールから就活関連のものを自動判別し、企業名・選考種別・必要なアクション・締切を自動抽出して就活の進捗を一元管理します。
さらに、**就活と無関係なメールは自動的に破棄され、就活メールだけがクリーンに整理される**ため、不要な情報に気を取られず管理が可能です。抽出された情報だけでなく、**実際のメール本文もUKABU上で確認**できます。

## 主な機能

### メール自動取り込み
メールを転送するだけで自動的にシステムに取り込まれます。就活関連メールかどうかはAIが自動判別し、**非関連メールは自動で破棄される**ため、ユーザーが選別する必要はありません。

### AI解析
AIがメール内容を解析し、以下の情報を自動抽出します：
- 企業名
- 選考種別（本選考/インターン）
- 必要なアクション（ES提出、面接参加など）
- 締切日時

解析された情報に加えて、**メール本文の全文をUKABU内で確認できる**ため、アプリ内で完結した情報収集が可能です。

### タスク管理
抽出されたアクションは自動的にタスクとして登録され、締切順に管理できます。AIがメールの内容から「このアクションは既に完了している」と判断した場合、自動的に完了状態としてマークされます。誤判定があった場合はユーザー自身で完了状態に変更できます。

### 企業管理
- **優先度の並び替え**: ドラッグ&ドロップで企業の優先順位を自由に変更
- **選考種別の分類**: 本選考とインターン選考をタブで切り替えて表示
- **複数の並び順**: 締切順・更新日順・優先度順で企業を並び替え可能

### 進捗可視化
各企業の選考状況を一覧で確認でき、次に取るべきアクションと締切が一目で分かります。

## 使い方

1. **アカウント作成**: メールアドレスとパスワードでアカウントを作成
2. **メール転送設定**: 受信メールを専用のアドレスに転送するよう設定
3. **自動管理開始**: 転送されたメールは自動的に解析され、企業・アクション情報として整理されます。
　　**不要なメールは自動破棄され、必要なメールだけがUKABU内に残ります。**
4. **進捗管理**: Webアプリで企業一覧やメール本文を確認。アクションの完了状態はAIが自動判定し、必要に応じてユーザーが修正できます。
